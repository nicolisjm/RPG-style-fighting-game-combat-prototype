# Input Map Setup

The player movement script uses Godot’s default **UI** actions for movement:

- `ui_left`, `ui_right`, `ui_up`, `ui_down` (often mapped to A, D, W, S or arrow keys)

For skills (used when combat is added), use:

| Action   | Key / Button        |
|----------|----------------------|
| `skill_1`| Left Mouse Button    |
| `skill_2`| Right Mouse Button   |
| `skill_3`| 1                    |
| `skill_4`| 2                    |
| `skill_5`| 3                    |
| `skill_6`| 4                    |

Add these in **Project → Project Settings → Input Map** if they are not already there.
